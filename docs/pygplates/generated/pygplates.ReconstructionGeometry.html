pygplates.ReconstructionGeometry¶

class pygplates.ReconstructionGeometry¶

Bases: Boost.Python.instance

The base class inherited by all derived reconstruction geometry classes.

The list of derived classes is:

  • ReconstructedFeatureGeometry
  • ReconstructedMotionPath
  • ReconstructedFlowline
  • ResolvedTopologicalLine
  • ResolvedTopologicalBoundary
  • ResolvedTopologicalNetwork
